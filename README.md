# TaskTracker.UI

The UI for LexisNexis Task Tracker to track the tasks.
Task Tracker UI

UI-Project TaskTracker.UI

Description

This is the frontend UI for the application built with Angular 18 and Node.js v18.19.1.
It contains the main app, feature modules, shared components, and core services.

Features

Modular Angular architecture (core, shared, features, components)

Reusable components and services

HTTP interceptors

Prerequisites global installations

Node.js v18.19.1

Angular CLI compatible with Angular 18 (npm install -g @angular/cli)

npm package manager

Installation

# Clone the repository

git clone https://github.com/sirsbusiso/TaskTracker.UI.git

# Install dependencies

npm install

# Run the development server

ng serve

# Open in browser

http://localhost:4200
